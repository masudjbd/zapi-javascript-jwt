# zapi-javascript-jwt

## Create JWT for JWT Authentication

- Create JWT token by using javascript.


## Define following parameters [index.html]

- username
- baseurl
- access key
- secret key
- http method
- api url
- jwt expire time.


and click to [Generate JWT]
